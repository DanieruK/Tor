Bei einer VPN handelt es sich um ein geschlossenes logisches Netzwerk, bei dem die Teilnehmer räumlich voneinander getrennt über einen IP-Tunnel eine Verbindung haben.

Mithilfe einer VPN kann man unterwegs auf sein lokales oder privates Netzwerk zugreifen. Nützlich ist dies zum Beispiel bei Unternehmen, denn oftmals sind Programme auf der Arbeit nicht öffentlich verfügbar.

Auch für den Privatgebrauch können VPNs nützlich sein. Möchte man zum Beispiel im Internet auf Preisangebote eines anderen Landes zugreifen, benutzt man eine VPN. Dabei wird Ihre Verbindung über einen Server im jeweiligen Land laufen, weshalb angenommen wird, dass man sich in diesem Land befindet.
