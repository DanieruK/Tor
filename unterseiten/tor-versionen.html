Hilfsprogramme

Um auf das Tor Netzwerk zugreifen zukönnen braucht man ein Hilfsprogramm, hier stehen einem unterschiedliche Zugangspunkte zur Verfügung.

Platzhalter

Tor Browser

Platzhalter

Der Tor Browser ist ein Paket aus dem Tor Client sowie einer modifizierten Version des Browsers Mozilla Firefox. Durch die auf Kompatiblität ausgelegte Konfiguration, ist es auch möglich den Tor Browser portabel von einem Wechseldatenträger aus zu starten.

Platzhalter

Tor Messenger

Platzhalter

Tor Messenger wurde am 29. Oktober 2015 veröffentlicht und seit März 2018 wieder eingestellt. Dabei handelte es sich um einen Messenger der auf Instantbird bassiert und hat dabei Tor zur Anonymisierung integriert.

Platzhalter

Orbot

Platzhalter

Orbot ist ein Quelloffener Proxy für Android ab der Version 2.0. Bei Geräten mit Root-Brechtigungen kann der ganze Internetverkehr durch das Tor.Netzwerk geleitet werden. Bei Geräten ohne dieser Brechtigung wird zusätzlich der Browser Orfox benötigt.

Platzhalter

Orfox

Platzhalter

Orfox ist speziell für Android und funktioniert mit Orbot zusammen. Dieser ist ein Quelloffener Browser der großen Wert auf Privatsphäre legt. Am 21. Mai 2019 wurde die erste stabile Version von Orfox veröffentlicht und die bisherige aplha Verison wurde eingestellt.

Platzhalter

Onion Browser

Platzhalter

Der Onion Browser ist für iOS Geräte aber der Version 5.1 bestimmt, er führt die Seitenabrufe durch das Tor-Netzwerk durch. Den Onion Browser kann man im Apple Store laden und auf GitHub findet man kostenlos den entsprechenden Quellcode
